# Sample input documents

Drop sample PDFs and images here for use by the examples (`bulk-tag-folder`, `llamaindex-reader`, etc.). Recommended set:

- `invoice.pdf` — a generic vendor invoice (single page, line items)
- `receipt.jpg` — a phone-camera photo of a paper receipt
- `contract.pdf` — a multi-page agreement

> _Note for maintainers:_ keep these realistic but **non-confidential**. No real vendor names, no real customer data, no PII. Either fully synthetic or sourced from public datasets.

When you add files here, also drop the corresponding FileTag JSON responses into [`../sample-output/`](../sample-output) so contributors can see the expected output shape without spending free-tier credits.
